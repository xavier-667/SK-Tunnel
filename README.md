# SK Tunnel

[![API](https://img.shields.io/badge/API-21%2B-brightgreen.svg?style=flat)]()
[![License](https://img.shields.io/badge/license-GPL--3.0-orange.svg)]()

SK Tunnel is a powerful Android VPN client built with sing-box and Xray-core technologies.

Built by Surprise Khoza.

## Features

- Fast and secure VPN connection
- Modern and user-friendly interface
- Multiple protocol support
- Built-in tunneling tools
- Subscription import support
- Custom configuration support
- Lightweight and optimized performance
- Advanced routing support

## Supported Protocols

- SOCKS5
- HTTP(S)
- SSH
- Shadowsocks
- VMess
- VLESS
- Trojan
- WireGuard
- Hysteria 1/2
- TUIC
- ShadowTLS
- AnyTLS

## Subscription Support

Supports importing:
- Clash formats
- Shadowsocks formats
- v2rayN formats
- sing-box outbound formats

## Requirements

- Android 5.0 or higher
- Internet connection
- VPN permission enabled

## Core Technologies

- sing-box
- Xray-core
